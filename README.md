# Noi’s Cozy Cup 

Noi’s Cozy Cup is a dynamic café website developed using Python and Django as part of a Web Development training program.

##  Features

- Dynamic coffee menu
- Careers page with job application model
- Contact page
- Location page
- Admin panel for managing content
- Structured Django project architecture

##  Technologies Used

- Python
- Django
- HTML5
- CSS3
- JS
- SQLite

## Project Structure

The project follows Django’s standard structure with:
- Apps (cafe)
- Templates
- Static files
- Migrations
- Media handling

##  How to Run the Project

1. Clone the repository
2. Create a virtual environment
3. Install dependencies
4. Run migrations:
   python manage.py migrate
5. Start the server:
   python manage.py runserver
